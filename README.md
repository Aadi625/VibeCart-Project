Overview

VibeCart is a full-stack e-commerce web application designed to provide a seamless online shopping experience.
It enables users to browse products, view detailed information, and interact with a modern, responsive interface.
The project demonstrates end-to-end development skills, including frontend UI design, backend development, and API integration.

Features

Browse and explore products
Product search and filtering (if implemented)
Detailed product pages
Dynamic data rendering from backend
Responsive and modern UI
REST API integration
Scalable project structure
